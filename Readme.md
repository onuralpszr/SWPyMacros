## Installation

* Requirements : Solidworks API, Python 3.7 (Tested on 3.7.3). This is only works on Windows

1. Method

```sh
# Pip Way
pip install -r requirements.txt
```
2. Method

```sh
# Pipenv Way
pip install pipenv
pipenv install
# Start env 
pipenv shell 
```

## Usage

```sh
# Make sure you're in virtualenv !
python hello_python_world.py
```
